Code snippets

Op de uitlegpagina's op deze website zijn code snippets zichtbaar. Dat zijn blokken tekst met daarin .html-code die mooi is opgemaakt zodat deze beter leesbaar is.

Deze code snippets zijn niet toegevoegd/opgemaakt met behulp van Boostrap maar met behulp van Prism.js.

Enkele voorbeelden

Op de andere pagina's worden alleen HTML- en JavaScript-snippets getoond. Maar stukken Python- of Java-code kunnen ook mooi worden opgemaakt:

          
print("Hello world!")
      
          
public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello World!");
    }
}
Hoe werkt dit?

Op de website van Prism.js kan je selecteren welke programmeertalen je wil kunnen opmaken en kan je vervolgens een .js- en .css-bestand downloaden. Plaats deze in respectievelijk de (nieuw aan te maken) ./scripts en ./css-mappen op je webserver.

Voeg vervolgens in de <head> van je .html-pagina het volgende toe zodat de stylesheet beschikbaar is: <link href="./css/prism.css" rel="stylesheet" />. Voeg ook de JavaScript toe door de volgende code onderaan de <body> te plakken, bijvoorbeeld onder de Bootstrap-JavaScript: <script src="scripts/prism.js"></script>

Het opmaken van een stukje code doe je vervolgens als volgt. Let op dat in de code die je wil opmaken de tekens '<' en '>' (waarmee HTML-tags worden geopend en gesloten) zijn vervangen door &lt; en &gt;. Als je dit niet doet en bijvoorbeeld een stukje HTML wil opmaken, wordt die code daadwerkelijk gebruikt op de pagina!

          
<pre>
    <code class="language-html">
       &lt;!DOCTYPE html&gt;
       &lt;html lang="nl"&gt;
       &lt;head&gt;
          &lt;meta charset="UTF-8"&gt;
          &lt;title&gt;Hello world&lt;/title&gt;
       &lt;/head&gt;
       &lt;body&gt;
          &lt;h1&gt;Hello world&lt;/h1&gt;
       &lt;/body&gt;
       &lt;/html&gt;
    </code>
</pre>
