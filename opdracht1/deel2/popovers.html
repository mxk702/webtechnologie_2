Pop-overs

Een pop-over lijkt op een tooltip; het is een pop-up die verschijnt wanneer de gebruiker op een element klikt. Het verschil is dat de popover veel meer inhoud kan bevatten dan een tooltip.

Wil je trouwens een voorbeeld zien van een tooltip? Houd je muis dan eens boven dit woord.

Over tot de orde van de dag, deze pagina gaat over pop-overs...

Voorbeelden van pop-overs

Hieronder staat een voorbeeld van hoe een pop-over eruit zou kunnen zien.

Toevoegen van pop-overs

Het toevoegen van een pop-over bestaat uit twee stappen. Stap één: toevoegen van een script. Voeg onderaan de body van je .html-webpagina een <script>-blok toe en zet daarin de volgende code:

        
var popoverTriggerList = [].slice.call(document.querySelectorAll('[data-bs-toggle="popover"]'))
var popoverList = popoverTriggerList.map(function (popoverTriggerEl) {
    return new bootstrap.Popover(popoverTriggerEl)
})

Stap twee: een pop-over toevoegen aan een element. Je kan een pop-over toevoegen aan veel soorten elementen, maar in dit voorbeeld voegen we een pop-over toe aan een knop. Onderstaande code is de code voor de pop-over die is verbonden aan de blauwe knop op deze pagina.

        
<button type="button" class="btn btn-primary" data-bs-toggle="popover" title="Dit is een pop-over" data-bs-content="Zoals je ziet kan je hier veel meer tekst in kwijt dan in een tooltip! Als je weer op de knop klikt, verdwijn ik.">Klik op mij</button>
