Validatie

Met validatie kan feedback worden gegeven aan gebruikers wanneer zij een formulier invullen. Probeer het onderstaande formulier maar eens te versturen zonder dat er iets is ingevuld en vul vervolgens één voor een de velden in.

Voorbeeldformulier
Correct ingevuld!
Voer een naam in.
@
Correct ingevuld!
Voer een e-mailadres in.
Je moet akkoord gaan.
Hoe werkt dit?

Door aan een <form> de code onderstaande code toe te voegen...

        
<form class="row g-3 needs-validation" novalidate action="./validation.html">

... Zorg je ervoor dat Bootstrap meldingen weergeeft als de validatie niet is geslaagd én zorg je ervoor dat de browser dit zelf niet doet (anders krijg je dubbele meldingen te zien).

Om een validatiemelding in te stellen voor een vraag in een form, voeg je aan de vraag een invalid-feedback en/of valid-feedback-<div> toe met daarin de melding die je wil tonen. Bijvoorbeeld:

        
<div class="invalid-feedback">
    Voer een e-mailadres in.
</div>

De form die op deze pagina zichtbaar is, ziet er dus zo uit:

        
<form class="row g-3 needs-validation" novalidate action="./validation.html">
    <div class="col-md-4">
      <label for="validationCustom01" class="form-label">Naam</label>
      <input type="text" class="form-control" id="validationCustom01" required>
      <div class="valid-feedback">
        Correct ingevuld!
      </div>
      <div class="invalid-feedback">
        Voer een naam in.
      </div>
    </div>
    <div class="col-md-4">
      <label for="validationCustomUsername" class="form-label">E-mailadres</label>
      <div class="input-group has-validation">
        <span class="input-group-text" id="inputGroupPrepend">@</span>
        <input type="email" class="form-control" id="validationCustomUsername" aria-describedby="inputGroupPrepend" required>
        <div class="valid-feedback">
          Correct ingevuld!
        </div>
        <div class="invalid-feedback">
          Voer een e-mailadres in.
        </div>
      </div>
    </div>
    <div class="col-12">
      <div class="form-check">
        <input class="form-check-input" type="checkbox" value="" id="invalidCheck" required>
        <label class="form-check-label" for="invalidCheck">
          Ik ga akkoord
        </label>
        <div class="invalid-feedback">
          Je moet akkoord gaan.
        </div>
      </div>
    </div>
    <div class="col-12">
      <button class="btn btn-primary" type="submit">Submit form</button>
    </div>
</form>
