Buttons

Buttons zijn knoppen waar je op kan klikken. Als je op een knop klikt, kan je bijvoorbeeld naar een andere pagina worden gestuurd of kan een formulier worden verstuurd.

Standaard HTML-knoppen

Normale HTML-knoppen, zonder styling dus, zien er nogal saai uit. Kijk maar:

Bootstrap-knoppen

... Daarentegen zien knoppen die met Bootstrap zijn gestyled er een stuk beter uit!

Toevoegen van buttons

Hoe voeg je een button toe? Simpel, bovenstaande buttons voeg je zo toe:

            
<button class="btn btn-primary">Knop</button>
<button class="btn btn-success">Goed</button>
<button class="btn btn-danger">Niet klikken</button>
<button class="btn btn-warning">Let op</button>
